A Minecraft Forge mod for Minecraft 1.20.1 that automates basic in-game tasks through configurable hotkeys and a finite state machine (FSM) architecture.

⚠️ **This project is still under development.**

Features, behavior, and implementation details may change as development progresses.

## Goals

- Automate repetitive in-game actions
- Provide a clean and extensible FSM-based framework
- Improve reliability through structured state management
- Add support for more complex automation workflows
